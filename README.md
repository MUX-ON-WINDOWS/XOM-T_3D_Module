## Installation
This how you install Three.js locally. 

For more information go to: https://threejs.org/docs/index.html?q=in#manual/en/introduction/Installation

````
# three.js
npm install --save three

# vite
npm install --save-dev vite
````

Launch:
````
npx vite
````
